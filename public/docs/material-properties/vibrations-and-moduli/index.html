    Vibrations and Moduli #

    The relationship between vibrational behavior and elastic moduli is fundamental to impulse excitation testing. Understanding this relationship is crucial for accurate property determination and material characterization.

    Physical Principles #

    Vibration Theory #

    When a material specimen is excited by an impulse, it vibrates at its natural frequencies. These frequencies are determined by:

    • Material properties (elastic moduli, density)
    • Geometry (dimensions, shape)
    • Boundary conditions (support configuration)

    Natural Frequencies #

    The natural frequencies of a vibrating system are the frequencies at which the system oscillates when disturbed from equilibrium. For impulse excitation testing:

    • First flexural mode - Primary bending vibration
    • First torsional mode - Primary twisting vibration
    • Higher modes - Additional vibration modes

    Mathematical Relationships #

    Flexural Vibration #

    For a rectangular bar in flexural vibration, the natural frequency is related to Young’s modulus by:

    f_f = (k_f / 2π) * √(E / ρ) * (t / L²)

    Where:

    • f_f = flexural frequency
    • k_f = frequency factor (depends on mode and boundary conditions)
    • E = Young’s modulus
    • ρ = density
    • t = thickness
    • L = length

    Torsional Vibration #

    For torsional vibration, the natural frequency relates to shear modulus:

    f_t = (k_t / 2π) * √(G / ρ) * (1 / L)

    Where:

    • f_t = torsional frequency
    • k_t = torsional frequency factor
    • G = shear modulus

    Poisson’s Ratio #

    Poisson’s ratio is calculated from the relationship between Young’s and shear moduli:

    ν = (E / 2G) - 1

    Frequency Factors #

    Flexural Frequency Factors #

    Rectangular bars:

    • First mode: k_f = 1.506
    • Second mode: k_f = 2.500
    • Higher modes: k_f increases

    Cylindrical bars:

    • First mode: k_f = 1.571
    • Second mode: k_f = 2.500
    • Higher modes: k_f increases

    Torsional Frequency Factors #

    Rectangular bars:

    • First mode: k_t = 1.000
    • Second mode: k_t = 2.000
    • Higher modes: k_t increases

    Cylindrical bars:

    • First mode: k_t = 1.000
    • Second mode: k_t = 2.000
    • Higher modes: k_t increases

    Material Properties #

    Young’s Modulus #

    Physical significance:

    • Resistance to axial deformation
    • Primary stiffness parameter
    • Temperature dependent
    • Process dependent

    Typical values:

    • Metals: 50-400 GPa
    • Ceramics: 100-500 GPa
    • Polymers: 0.1-10 GPa
    • Composites: 10-200 GPa

    Shear Modulus #

    Physical significance:

    • Resistance to shear deformation
    • Related to Young’s modulus
    • Important for torsion applications
    • Material dependent

    Typical values:

    • Metals: 20-150 GPa
    • Ceramics: 40-200 GPa
    • Polymers: 0.03-3 GPa
    • Composites: 3-80 GPa

    Poisson’s Ratio #

    Physical significance:

    • Ratio of lateral to axial strain
    • Material constant
    • Range: 0 to 0.5 for most materials
    • Related to E and G

    Typical values:

    • Metals: 0.25-0.35
    • Ceramics: 0.15-0.25
    • Polymers: 0.30-0.45
    • Composites: 0.20-0.40

    Measurement Considerations #

    Sample Geometry #

    Rectangular specimens:

    • Length: 3-5 times width
    • Width: 2-4 times thickness
    • Thickness: As thin as possible
    • Supports at 0.224L from ends

    Cylindrical specimens:

    • Length: 4-5 times diameter
    • Diameter: 10-50 mm typical
    • Supports at 0.224L from ends
    • Uniform cross-section

    Support Configuration #

    Flexural mode:

    • Supports at 0.224L from each end
    • Minimal contact area
    • Low damping materials
    • Stable positioning

    Torsional mode:

    • Supports at center
    • Allows free rotation
    • Minimal flexural coupling
    • Stable positioning

    Property Calculation #

    Young’s Modulus #

    From flexural frequency:

    E = (4π² × ρ × L⁴ × f_f²) / (k_f × t²)

    Calculation steps:

    1. Measure specimen dimensions
    2. Determine density
    3. Measure flexural frequency
    4. Apply frequency factor
    5. Calculate Young’s modulus

    Shear Modulus #

    From torsional frequency:

    G = (4π² × ρ × L² × f_t²) / k_t

    Calculation steps:

    1. Measure specimen dimensions
    2. Determine density
    3. Measure torsional frequency
    4. Apply frequency factor
    5. Calculate shear modulus

    Poisson’s Ratio #

    From E and G:

    ν = (E / 2G) - 1

    Calculation steps:

    1. Calculate Young’s modulus
    2. Calculate shear modulus
    3. Apply relationship
    4. Verify reasonable value

    • Impacts accuracy

    Thickness:

    • Affects frequency calculations
    • Influences mode shapes
    • Impacts accuracy
    • Process dependent

    Material Properties #

    Density:

    • Required for calculations
    • Affects frequency
    • Temperature dependent
    • Process dependent

    Isotropy:

    • Assumption of uniform properties
    • May not hold for all materials
    • Affects calculations
    • Requires verification

    Environmental Conditions #

    Temperature:

    • Affects material properties
    • Influences dimensions
    • Changes frequency
    • Requires control

    Humidity:

    • May affect damping
    • Influences properties
    • Changes frequency
    • Requires control
