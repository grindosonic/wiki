    Running Impulse Excitation Measurements #

    This guide provides detailed procedures for executing impulse excitation measurements, from sample placement to data analysis and result interpretation.

    Pre-Measurement Checklist #

    Sample Preparation #

    • Specimen dimensions measured accurately
    • Mass determined and recorded
    • Surface finish verified
    • Sample identification confirmed
    • Temperature stabilized

    Equipment Verification #

    • System calibrated and ready
    • Supports positioned correctly
    • Sensor functioning properly
    • Software configured
    • Environmental conditions stable

    Measurement Procedure #

    1. Sample Placement #

    Positioning the specimen:

    1. Place specimen on supports
    2. Ensure proper contact with supports
    3. Verify specimen is level and stable
    4. Check for any rocking or movement
    5. Confirm support positioning (0.224L for flexural)

    Support verification:

    • Minimal contact area
    • No excessive pressure
    • Stable positioning
    • Consistent with previous measurements

    2. Sensor Positioning #

    Accelerometer placement:

    1. Position near anti-node for maximum signal
    2. For flexural mode: center of specimen
    3. For torsional mode: edge of specimen
    4. Ensure secure attachment
    5. Verify electrical connections

    Signal verification:

    • Check signal level
    • Verify frequency response
    • Confirm proper grounding
    • Test data acquisition

    3. Excitation Process #

    Manual excitation:

    1. Position excitation device near anti-node
    2. Apply controlled impulse
    3. Ensure perpendicular strike
    4. Use consistent force level
    5. Allow specimen to vibrate freely

    Automated excitation:

    1. Program excitation parameters
    2. Set force level and duration
    3. Position excitation device
    4. Execute excitation sequence
    5. Monitor signal quality

    4. Data Acquisition #

    Signal capture:

    1. Start data acquisition
    2. Apply excitation
    3. Record vibration signal
    4. Monitor signal quality
    5. Stop acquisition when complete

    Signal quality checks:

    • Adequate signal amplitude
    • Clear frequency peaks
    • Minimal noise
    • Proper frequency range
    • Sufficient signal duration

    5. Signal Processing #

    FFT analysis:

    1. Apply window function (Hanning recommended)
    2. Perform FFT transformation
    3. Identify natural frequencies
    4. Measure peak amplitudes
    5. Calculate frequency resolution

    Peak identification:

    • First flexural mode (f_f)
    • First torsional mode (f_t)
    • Higher order modes (if needed)
    • Verify mode shapes
    • Check for mode coupling

    Data Analysis #

    Frequency Measurement #

    Peak detection:

    1. Locate frequency peaks
    2. Measure peak frequencies
    3. Verify peak sharpness
    4. Check for multiple peaks
    5. Record frequency values

    Quality assessment:

    • Peak sharpness (Q-factor)
    • Signal-to-noise ratio
    • Frequency stability
    • Mode separation
    • Consistency with theory

    Property Calculations #

    Young’s Modulus:

    E = (4π² × ρ × L⁴ × f_f²) / (k_f × t²)

    Shear Modulus:

    G = (4π² × ρ × L² × f_t²) / k_t

    Poisson’s Ratio:

    ν = (E / 2G) - 1

    Where:

    • E = Young’s modulus
    • G = Shear modulus
    • ν = Poisson’s ratio
    • ρ = Density
    • L = Length
    • t = Thickness
    • f_f = Flexural frequency
    • f_t = Torsional frequency
    • k_f, k_t = Frequency factors

    Uncertainty Analysis #

    Sources of uncertainty:

    • Dimensional measurements
    • Mass determination
    • Frequency measurement
    • Temperature effects
    • Environmental conditions

    Uncertainty calculation:

    1. Identify uncertainty sources
    2. Quantify each contribution
    3. Combine uncertainties
    4. Report with confidence level
    5. Document assumptions

    Quality Control #

    Repeatability Testing #

    Multiple measurements:

    1. Perform 5-10 measurements
    2. Calculate mean and standard deviation
    3. Check for outliers
    4. Verify consistency
    5. Report statistical parameters

    Acceptance criteria:

    • Standard deviation < 1% of mean
    • No significant outliers
    • Consistent frequency values
    • Stable material properties

    Validation Procedures #

    Reference materials:

    1. Test certified reference specimens
    2. Compare with certified values
    3. Verify measurement accuracy
    4. Document any deviations
    5. Adjust system if needed

    Cross-validation:

    • Compare with other methods
    • Verify with literature values
    • Check temperature effects
    • Validate material behavior

    Troubleshooting #

    Common Issues #

    Poor signal quality:

    • Check sensor positioning
    • Verify connections
    • Reduce noise sources
    • Improve excitation
    • Check specimen condition

    Inconsistent frequencies:

    • Verify support positioning
    • Check specimen geometry
    • Control environmental conditions
    • Improve excitation consistency
    • Check for specimen damage

    Missing modes:

    • Verify excitation location
    • Check support configuration
    • Increase excitation force
    • Improve signal processing
    • Check specimen geometry

    Diagnostic Procedures #

    Signal analysis:

    1. Examine time domain signal
    2. Check frequency spectrum
    3. Identify noise sources
    4. Verify mode shapes
    5. Optimize parameters

    System verification:

    1. Test with reference specimen
    2. Check all connections
    3. Verify software operation
    4. Calibrate sensors
    5. Update system parameters

    Data Recording #

    Measurement Log #

    Required information:

    • Sample identification
    • Date and time
    • Operator name
    • Environmental conditions
    • Equipment settings
    • Raw data files
    • Calculated properties
    • Uncertainty values

    File Organization #

    Data storage:

    • Raw time domain signals
    • Processed frequency spectra
    • Calculated properties
    • Measurement logs
    • Calibration data
    • Reference measurements

    Naming conventions:

    • Sample ID
    • Date/time stamp
    • Measurement type
    • Version number
    • File format

    Best Practices #

    Measurement Technique #

    1. Consistent procedure - Follow standard protocol
    2. Proper preparation - Ensure sample readiness
    3. Careful positioning - Accurate support and sensor placement
    4. Quality excitation - Consistent and appropriate force
    5. Thorough analysis - Complete data processing

    Quality Assurance #

    1. Regular calibration - Maintain system accuracy
    2. Reference testing - Validate with known materials
    3. Statistical analysis - Multiple measurements
    4. Documentation - Complete records
    5. Continuous improvement - Learn from experience

    Safety Considerations #

    1. Equipment safety - Follow manufacturer guidelines
    2. Sample handling - Avoid damage to specimens
    3. Environmental safety - Control hazards
    4. Data security - Protect measurement data
    5. Documentation - Maintain safety records

    Following these procedures will ensure accurate, reliable, and reproducible impulse excitation measurements for your specific application.
